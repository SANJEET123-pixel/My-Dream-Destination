# My Dream Destination - Japan

This project is a simple static website built with HTML and CSS.
The main page, `index.html`, presents information about Japan as a dream travel destination.

## What `index.html` Includes

- Page title and heading: **My Dream Destination: Japan**
- A short introduction about Japan
- A featured image (`assets/images/img1.jpg`)
- Top attractions list
- Things-to-do ordered list
- Attractions table by city
- External travel resource links
- Internal navigation to the contact page (`public/contact.html`)

## Styling

- Main stylesheet: `assets/css/style.css`

## How to Run

1. Open `index.html` in any web browser.
2. Or run with VS Code Live Server (if installed) for easier preview.

## Notes

- This is a beginner-friendly static web project.
- No build tools or dependencies are required.
